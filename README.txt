Developed a self-checking SystemVerilog verification environment for a synchronous SRAM, applying core UVM methodology principles.

The project includes: sram.sv, test.sv, testbench.sv, scoreboard.sv, coverage.sv, and assertions.sv.

Achieved 100% coverage. Please see the screenshot "coverage.png" to see the functional coverage.
